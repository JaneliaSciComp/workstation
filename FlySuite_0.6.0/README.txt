DESCRIPTION

JCVI VI Compute Server (VICS) is a collection of grid-enabled
bioinformatics tools with HTTP (browser) and Web Services (scriptable)
interfaces built on infrastructure.  Its functionality allows simple
execution of bioinformatics tools, as well as rapid implementation of
stable pipelines for high throughput production.

This project is available at: http://sourceforge.net/projects/jcvi-vics
